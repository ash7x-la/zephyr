# ZEPHYR - Advanced Autonomous Agentic AI

Zephyr is a high-precision, autonomous coding assistant built on the Antigravity Hybrid architecture. It merges deep project reasoning with a minimalist, premium TUI.

## Features
- ReAct Orchestration: Transparent <thought> and <action> logic.
- Hybrid Providers: Support for Gemini, OpenRouter, Claude, and Zephyr Free (Local Proxy).
- Persistent Memory: Integrated SOUL.md, MEMORY.md, and USER.md identity system.
- Async Stability: Powered by Textual for a smooth, flicker-free experience.
- Stealth Reasoning: Internal thoughts are hidden from the chat bubble to maintain a clean UI.
- Strict Professionalism: No emojis or emoticons in project outputs.

## Getting Started
1. Configure Environment:
   ```bash
   cp .env.example .env
   # Add your API Keys and DEEPSEEK_FREE_TOKEN
   ```
2. Launch Zephyr Free (Optional):
   ```bash
   ./run-zephyr-free.sh
   ```
3. Start the Assistant:
   ```bash
   python3 main.py
   ```
